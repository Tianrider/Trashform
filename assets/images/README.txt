PLACEHOLDER FOR ILLUSTRATION IMAGE

The image should depict hands holding a smartphone that is scanning a plastic bottle, similar to the one shown in the app design.

The image should be:
1. Transparent background (PNG format)
2. Centered composition
3. Clear and visible at various screen sizes
4. In a style that matches the app's green, eco-friendly theme
